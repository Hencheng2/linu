Command History

Common Linux Commands

  • ls - List directory contents FILE OPERATION
    List information about files and directories

    Usage: ls [options] [file|dir]

    ls -la /home/user

    Common options:
    -l: use a long listing format
    -a: show all files including hidden
    -h: human-readable file sizes

  • grep - Search text TEXT PROCESSING
    Search for patterns in files or input

    Usage: grep [options] pattern [file...]

    grep -r "password" /var/www/

    Common options:
    -r: recursive search
    -i: case-insensitive search
    -v: invert match (show non-matching lines)

  • chmod - Change file mode PERMISSIONS
    Change file permissions

    Usage: chmod [options] mode file

    chmod 755 script.sh

    Common options:
    -R: recursive change
    -v: verbose output

    Permission values:
    7: read, write, execute (rwx)
    6: read, write (rw-)
    5: read, execute (r-x)
    4: read only (r--)

Security & Penetration Testing Tools

  • nmap - Network Mapper NETWORK SCANNING
    Network discovery and security auditing tool

    Usage: nmap [scan type] [options] target

    nmap -sS -sV -O -T4 192.168.1.1/24

    Common options:
    -sS: TCP SYN stealth scan
    -sV: version detection
    -O: OS detection
    -A: aggressive scan (OS, version, script, traceroute)
    -p: specify ports (e.g., -p 80,443 or -p 1-1000)

    Example uses:
    • Discover live hosts on a network
    • Identify open ports and services
    • Detect operating systems and versions
    • Find vulnerabilities with NSE scripts

  • Maltego - OSINT & Forensics DATA MINING
    Interactive data mining tool for link analysis

    Usage: GUI-based tool, not command-line only

    maltego

    Key features:
    • Transformations to gather information from various sources
    • Visualization of relationships between entities
    • Extensive library of transforms for different data types

    Example uses:
    • Domain name research and DNS records
    • Social network relationships analysis
    • Email address investigation
    • Infrastructure mapping

  • recon-ng - Web Reconnaissance WEB RECON
    Full-featured web reconnaissance framework

    Usage: recon-ng

    recon-ng -w example_workspace

    Key features:
    • Module-based structure for reconnaissance tasks
    • Workspace organization for different projects
    • Database for storing results
    • API integration with various services

    Example workflow:
    1. Create a workspace
    2. Add domains or companies
    3. Load and execute modules for specific recon tasks
    4. Export results to reports

  • nikto - Web Vulnerability Scanner WEB SCANNING
    Comprehensive web server scanner

    Usage: nikto [options] -h host

    nikto -h https://example.com -p 80,443 -C all

    Common options:
    -h: target host
    -p: target ports
    -C: display output in CSV format
    -T: tuning (e.g., 0-9 for scan intensity)
    -o: output to file

    Scans for:
    • Server misconfigurations
    • Default files and programs
    • Outdated server components
    • Common vulnerabilities

  • metasploit - Penetration Testing EXPLOITATION
    Advanced penetration testing framework

    Usage: msfconsole

    msfconsole -q

    Key components:
    • Exploits: code that uses vulnerabilities
    • Payloads: code that runs after exploitation
    • Auxiliary modules: scanning, fuzzing, sniffing
    • Post-exploitation modules

    Basic workflow:
    1. use [exploit_path]
    2. set payload [payload_name]
    3. set RHOSTS [target]
    4. set LHOST [attacker_ip]
    5. exploit

  • sqlmap - SQL Injection Tool SQL INJECTION
    Automatic SQL injection and database takeover tool

    Usage: sqlmap [options] -u url

    sqlmap -u "http://example.com/page.php?id=1" --dbs

    Common options:
    -u: target URL
    --dbs: enumerate databases
    --tables: enumerate tables
    --columns: enumerate columns
    --dump: dump table contents
    --os-shell: prompt for an interactive OS shell

    Example uses:
    • Test and exploit SQL injection flaws
    • Extract database structure and data
    • Gain file system access
    • Execute commands on the database server

  • John the Ripper - Password Cracker PASSWORD CRACKING
    Fast password cracker

    Usage: john [options] password_file

    john --wordlist=/usr/share/wordlists/rockyou.txt hashes.txt

    Common options:
    --wordlist: specify wordlist file
    --format: specify hash format
    --rules: enable word mangling rules
    --show: show cracked passwords

    Example uses:
    • Crack Linux /etc/shadow passwords
    • Crack Windows LM/NTLM hashes
    • Crack password-protected archives
    • Crack WiFi WPA-PSK captures

  • hashcat - Advanced Password Recovery PASSWORD CRACKING
    World's fastest password cracker

    Usage: hashcat [options] hash|hashfile [wordlist|mask|directory]

    hashcat -m 0 -a 0 hashes.txt /usr/share/wordlists/rockyou.txt

    Common options:
    -m: hash type (0 for MD5, 1000 for NTLM)
    -a: attack mode (0 straight, 3 brute-force)
    -o: output file for recovered passwords
    --force: ignore warnings

    Advanced features:
    • GPU acceleration support
    • Distributed cracking networks
    • Rule-based attacks
    • Brute-force and mask attacks
